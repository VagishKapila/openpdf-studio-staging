================================================================================
CODE SIGNING SUPPORT
================================================================================

Windows Code Signing:
- Using signtool.exe from Windows SDK
- Supports DigiCert, Sectigo, Comodo certificates
- .pfx file method support
- Timestamp server integration (prevents expiration)
- PowerShell script automation

macOS Code Signing:
- Developer ID Application certificate
- Automatic during build
- Notarization support for App Store distribution
- Entitlements configuration included

Linux:
- No code signing required
- Package managers verify via checksums
- AppImage and Deb formats supported

================================================================================
SECURITY FEATURES
================================================================================

Content Security Policy (CSP):
- Default: same-origin scripts only
- Whitelisted CDNs: cdnjs.cloudflare.com, cdn.jsdelivr.net, unpkg.com
- Allowed domains: GitHub API (updates), document loading

File System Security:
- All file operations through Tauri sandbox
- User file selection via native dialogs
- No access to system files outside home directory

Code Execution:
- Process spawning for qpdf (PDF encryption)
- Only permitted system commands exposed

================================================================================
BUILD OUTPUT
================================================================================

Generated Installers:

Windows:
- .msi file (Windows Installer format)
- .nsis file (Traditional setup wizard)

macOS:
- .dmg file (Disk image, drag-and-drop installation)

Linux:
- .deb file (Debian/Ubuntu package)
- .AppImage file (Universal Linux executable)

Typical Sizes:
- Windows MSI: 80-120 MB
- macOS DMG: 90-130 MB
- Linux AppImage: 70-100 MB
- Linux Deb: 50-80 MB

QUICK START
================================================================================

1. Prerequisites:
   - Node.js v18+
   - Rust (latest stable)
   - Platform-specific dev tools

2. Installation:
   $ npm install

3. Development:
   $ npm run tauri:dev

4. Production Build:
   $ npm run tauri:build:release

5. Code Signing (if distributing):
   - See src-tauri/SIGNING_SETUP.md for detailed instructions
   - Windows: Configure certificate thumbprint
   - macOS: Setup Developer ID Application certificate
